# File-Handling-Project
"A simple Python-based CLI application to perform file operations like create, read, update, and delete using pathlib and os modules."

# 📂 File Handling System in Python

This is a simple **File Handling Project** built using Python.  
It allows users to perform basic file operations like:

- Create a file
- Read a file
- Update a file
- Delete a file


##  Features

✅ Create new files  
✅ Read existing files  
✅ Update file name or content  
✅ Append data to files  
✅ Delete files safely  
✅ Simple CLI-based interface  


##  Technologies Used

- Python 3
- pathlib module
- os module


## 📸 How It Works

1. Run the program  
2. Choose an option:
   - 1 → Create file  
   - 2 → Read file  
   - 3 → Update file  
   - 4 → Delete file  
3. Follow the instructions on screen
